# ABAP String Processing & Text Symbols Project

## Description
This project demonstrates key ABAP concepts:
- Translatable text using text symbols
- String processing using REPLACE()
- Built-in string functions (FIND, COUNT, SUBSTRING, etc.)
- Basic use of regular expressions (PCRE)

---

## Features
- Dynamic text replacement using placeholders
- Clean formatted console output
- Case conversion and string manipulation
- Regex-based search


---

## How to Run
1. Open the class in ABAP Development Tools (ADT)
2. Activate the class (`Ctrl + F3`)
3. Run as console application (`F9`)
4. View output in console



## Concepts Used
- Text Symbols (Translation-ready UI text)
- String Functions (`REPLACE`, `SUBSTRING`, `TO_UPPER`)
- Search Functions (`FIND`, `COUNT`)
- Regular Expressions (PCRE)
